iOS集成OpenCV.framework (c++) 注意事项

发表于 2018-06-23

iOS集成OpenCV.framework (c++)

  • target - buildSettings - Compile Sources As 设置为默认的（According toFile Type）

    • 原因：不要设置为 object C++ ，设置c++ 后，会对你的原有oc代码有冲突，很烦！

  • 将文件中的 .cpp 文件改为 .mm

  • 将使用到c++ 文件 的原有文件 .m 改为 .mm (XCode自动设置当前.m 文件为 objectc++ source)

  • 配合pch文件

  • 在.mm中导入c++ opencv 头文件时， 必须在最上面

  • 使用到自己写的c++ .h文件， 引用它 在需要使用的.mm 文件中， 属性的定义也必须在.mm 中， 不可在.h 中使用
